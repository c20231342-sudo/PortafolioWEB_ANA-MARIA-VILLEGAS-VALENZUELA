# 📖 Descripción

Este repositorio contiene el Portafolio de Evidencias desarrollado durante la materia de Desarrollo de Aplicaciones Web. En él se integran las prácticas, actividades, proyectos y evidencias realizadas a lo largo del curso, aplicando conocimientos relacionados con desarrollo web, bases de datos, APIs, WordPress y frameworks de desarrollo.

El objetivo del portafolio es documentar el aprendizaje adquirido durante el semestre mediante proyectos funcionales y documentación técnica.
