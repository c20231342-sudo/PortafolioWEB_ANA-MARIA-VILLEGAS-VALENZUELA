# 📖 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un sitio web tipo portafolio utilizando WordPress, con el objetivo de presentar información personal, habilidades, servicios y proyectos desarrollados durante la materia de Desarrollo de Aplicaciones Web.

El sitio fue diseñado con una estructura organizada, navegación intuitiva y una apariencia profesional utilizando las herramientas proporcionadas por WordPress.

---

# 🎯 Objetivo

Diseñar e implementar un sitio web tipo portafolio utilizando WordPress, aplicando conocimientos sobre:

- Instalación y configuración de WordPress.
- Creación y administración de páginas.
- Configuración de menús de navegación.
- Personalización de temas.
- Uso de plugins.
- Organización de contenido web.
- Diseño responsivo y experiencia de usuario.

# 📧 Plugins Utilizados

## Contact Form 7

Plugin utilizado para crear formularios de contacto personalizados dentro del sitio web.

### Funciones:

- Creación de formularios.
- Envío de mensajes.
- Integración sencilla mediante shortcode.

## Flamingo

Plugin complementario utilizado para almacenar los mensajes enviados desde el formulario de contacto.

### Funciones:

- Registro de mensajes recibidos.
- Almacenamiento dentro de WordPress.
- Seguimiento de solicitudes de contacto.

# ⚙ Instalación del Proyecto

## 1. Instalar XAMPP

Descargar e instalar XAMPP para habilitar Apache y MySQL.

## 2. Iniciar Servicios

Desde el panel de control de XAMPP iniciar:

- Apache
- MySQL

## 3. Instalar WordPress

Copiar los archivos de WordPress dentro del directorio:

```text
C:\xampp\htdocs\
```

## 4. Crear Base de Datos

Abrir phpMyAdmin y crear una base de datos para WordPress.

## 5. Configurar WordPress

Completar el asistente de instalación configurando:

- Nombre del sitio.
- Usuario administrador.
- Contraseña.
- Correo electrónico.
